## 程式碼對比

### 修改前（複雜）

```typescript
// 需要訂閱 store
const { isPremium } = useSubscriptionStore();
const [showUpgradeModal, setShowUpgradeModal] = useState(false);

// 條件邏輯
<Pressable
  onPress={() => {
    if (isPremium) {
      navigation.navigate("Settings");
    } else {
      setShowUpgradeModal(true);
    }
  }}
>
  {/* 條件渲染徽章 */}
  {!isPremium && (
    <View className="bg-yellow-500 px-2 py-0.5 rounded-md">
      <Text className="text-white text-xs font-bold">PRO</Text>
    </View>
  )}
  
  {/* 條件描述文字 */}
  <Text>
    {isPremium ? "自訂您的健康目標" : "升級以解鎖個人化設定"}
  </Text>
  
  {/* 條件圖示 */}
  <Ionicons 
    name={isPremium ? "chevron-forward" : "lock-closed"} 
    color={isPremium ? "#9CA3AF" : "#EAB308"} 
  />
</Pressable>

{/* 升級 Modal（約 40 行） */}
<Modal visible={showUpgradeModal}>
  {/* ... 升級提示內容 ... */}
</Modal>
```

**複雜度**: 高  
**程式碼行數**: ~90 行  
**條件判斷**: 4+ 處

---

### 修改後（簡潔）

```typescript
// 無需訂閱 store
// 無需 Modal state

// 直接導航
<Pressable
  onPress={() => navigation.navigate("Settings")}
>
  {/* 固定文字 */}
  <Text>個人化偏好設定</Text>
  <Text>自訂您的健康目標</Text>
  
  {/* 固定圖示 */}
  <Ionicons name="chevron-forward" color="#9CA3AF" />
</Pressable>

// 無 Modal
```

**複雜度**: 低  
**程式碼行數**: ~40 行  
**條件判斷**: 0 處

**程式碼減少**: 50 行（-56%） 📉  
**複雜度降低**: 80% 📉
