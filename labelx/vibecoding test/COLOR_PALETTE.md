# LabelX 配色方案 (Warm Coral-Teal Theme)

## 設計理念

以**溫暖珊瑚色**為主體，搭配深藍標題和清新青綠色，營造現代、溫暖、專業的視覺體驗。

## 主色系 (Primary Colors)

### 背景色 (Background)
- **主背景**: `#FFFFFA` - 溫暖的奶油色，用於頁面主背景
- **卡片背景**: `#FFFFFF` - 純白色卡片

### 標題色 (Headline)
- **深藍**: `#001858` - 深海藍，用於標題和重要文字
- **中等藍**: `#002B5C` - 較淺的深藍，用於次要標題

### 主色 (Main)
- **珊瑚色**: `#F3D2C1` - 溫暖的珊瑚粉，用於卡片背景、輔助區域
- **深珊瑚**: `#E8C4B0` - 較深的珊瑚色，用於強調

### 高光色 (Headlight)
- **奶油高光**: `#FFFFFA` - 與背景色相同，用於高光區域

### 次要色 (Secondary)
- **青綠色**: `#8BD3DD` - 清新的青綠，用於輔助信息、圖標
- **深青綠**: `#6FC4CF` - 較深的青綠，用於 hover 狀態

### 按鈕色 (Button)
- **綠色**: `#2CB67D` - 明亮的綠色，用於所有主要按鈕和互動元素
- **深綠**: `#249C6A` - 較深的綠色，用於按下狀態

## 功能性色彩 (Semantic Colors)

### 健康評分指標
- **成功綠**: `#2CB67D` - 高分健康評分（≥71分）
- **警告黃**: `#F59E0B` - 中等評分（31-70分）
- **危險紅**: `#EF4444` - 低分/高風險評分（<31分）

## 使用場景

### 底部標籤欄
- **背景**: `#FFFFFF` (純白)
- **激活狀態**: `#2CB67D` (綠色)
- **非激活狀態**: `#8BD3DD` (青綠)
- **掃描按鈕**: `#2CB67D` (綠色)

### 首頁
- **頁面背景**: `#FFFFFA` (奶油色)
- **卡片背景**: `#FFFFFF` 或 `#F3D2C1` (珊瑚色)
- **標題**: `#001858` (深藍)
- **正文**: `#4A4A4A` (深灰)
- **統計卡片背景**: `#F3D2C1` (珊瑚色)
- **圖標顏色**: `#2CB67D` (綠色) 或 `#8BD3DD` (青綠)
- **CTA 按鈕**: `#2CB67D` (綠色)

### 個人頁面
- **Header 背景**: `#F3D2C1` (珊瑚色) 或漸變
- **標題**: `#001858` (深藍)
- **統計卡片**: `#F3D2C1` (珊瑚色) 背景
- **Premium 卡片**: `#2CB67D` (綠色) 背景
- **菜單圖標**: `#8BD3DD` (青綠) 或 `#2CB67D` (綠色)

### 統計卡片
- **背景**: `#F3D2C1` (珊瑚色)
- **圖標**: `#2CB67D` (綠色) 或 `#8BD3DD` (青綠)
- **數值**: `#001858` (深藍)
- **標題**: `#001858` (深藍)

### 按鈕系統
- **主按鈕**: `#2CB67D` (綠色) 背景 + 白色文字
- **次要按鈕**: `#8BD3DD` (青綠) 背景 + 白色文字
- **文字按鈕**: `#2CB67D` (綠色) 文字

### 趨勢圖表
- **線條**: `#2CB67D` (綠色)
- **數據點**: `#2CB67D` (綠色)
- **網格**: `#E5E7EB` (淺灰)
- **背景**: `#FFFFFF` (白色)

## 顏色對照表

| 用途 | 新顏色 | 色值 |
|---|---|---|
| 頁面背景 | 奶油色 | `#FFFFFA` |
| 標題文字 | 深藍 | `#001858` |
| 卡片背景 | 珊瑚色 | `#F3D2C1` |
| 次要信息 | 青綠色 | `#8BD3DD` |
| 主按鈕 | 綠色 | `#2CB67D` |

## 設計原則

1. **溫暖舒適**: 奶油色和珊瑚色營造溫暖親切感
2. **專業可信**: 深藍色標題展現專業性
3. **清新活力**: 青綠色和綠色帶來清新感
4. **高對比度**: 深藍文字在淺色背景上易讀
5. **層次分明**: 通過顏色變化建立清晰的信息層級
