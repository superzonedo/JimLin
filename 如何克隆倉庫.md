# 如何在 Cursor 中克隆 GitHub 倉庫

## 方法 1：使用終端命令（最簡單）

1. 在 Cursor 中打開終端（Terminal）
   - 快捷鍵：`Ctrl + `` (反引號) 或 `Cmd + `` (Mac)
   - 或點擊選單：Terminal → New Terminal

2. 執行以下命令：
```bash
cd ~/Documents  # 或您想要保存項目的位置
git clone https://github.com/superzonedo/JimLin.git
cd JimLin
```

3. 在 Cursor 中打開項目：
   - File → Open Folder
   - 選擇剛才克隆的 `JimLin` 資料夾

## 方法 2：使用 Cursor 的 Source Control

1. 打開 Cursor
2. 點擊左側的 Source Control 圖示（或按 `Ctrl+Shift+G` / `Cmd+Shift+G`）
3. 點擊右上角的 "..." 選單
4. 選擇 "Clone" 或 "Clone Repository"
5. 輸入：`https://github.com/superzonedo/JimLin.git`
6. 選擇保存位置

## 方法 3：使用命令面板

1. 打開命令面板：
   - Windows/Linux: `Ctrl + Shift + P`
   - Mac: `Cmd + Shift + P`

2. 輸入以下任一命令：
   - `Git: Clone`
   - `Git: Clone Repository`
   - `Clone`

3. 輸入倉庫 URL：`https://github.com/superzonedo/JimLin.git`

4. 選擇保存位置

## 方法 4：直接從 GitHub 下載

1. 前往：https://github.com/superzonedo/JimLin
2. 點擊綠色的 "Code" 按鈕
3. 選擇 "Download ZIP"
4. 解壓縮 ZIP 文件
5. 在 Cursor 中打開解壓後的資料夾

## 方法 5：使用 GitHub Desktop（如果已安裝）

1. 打開 GitHub Desktop
2. 點擊 "File" → "Clone Repository"
3. 選擇 "URL" 標籤
4. 輸入：`https://github.com/superzonedo/JimLin.git`
5. 選擇本地保存位置
6. 點擊 "Clone"
7. 在 Cursor 中打開克隆的資料夾

## 推薦方法

**最推薦使用方法 1（終端命令）**，因為：
- 最簡單直接
- 不需要尋找選單
- 適用於所有系統
- 可以立即開始使用

## 克隆後需要做的事情

1. **安裝依賴**：
```bash
# 安裝根目錄依賴
npm install

# 安裝 functions 依賴
cd functions
npm install
cd ..

# 安裝 labelx 依賴（如果需要）
cd labelx
npm install
cd ..
```

2. **設置環境變數**：
   - 創建 `.env` 文件（如果有的話）
   - 設置 Firebase 配置
   - 設置 Google OAuth 憑證（從 Google Cloud Console 獲取）

3. **查看文檔**：
   - 閱讀 `GITHUB_UPLOAD_GUIDE.md`
   - 查看其他 `.md` 文件了解設置步驟

## 注意事項

⚠️ **重要**：
- 倉庫中已移除所有敏感資訊（Client ID、Secret 等）
- 需要從 Google Cloud Console 和 Firebase Console 獲取實際的憑證
- 請參考 `SETUP_FIREBASE_SECRETS.md` 等文檔來設置
